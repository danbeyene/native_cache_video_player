# Contributing to Native Cache Video Player

Thank you for your interest in contributing! We welcome bug reports, feature requests, and pull requests.

## Reporting Issues
Please submit issues via GitHub with:
1.  A descriptive title.
2.  Steps to reproduce the issue.
3.  Expected vs. actual behavior.
4.  Logs or screenshots if applicable.

## Pull Requests
1.  Fork the repository.
2.  Create a feature branch.
3.  Commit your changes.
4.  Push to your branch and submit a PR.
5.  Ensure all tests pass and `flutter analyze` shows no errors.

## Development Setup
1.  Clone the repo: `git clone https://github.com/danbeyene/native_cache_video_player.git`
2.  Install dependencies: `flutter pub get`
3.  Run the example app: `cd example && flutter run`
